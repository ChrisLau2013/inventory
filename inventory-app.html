Data Backup

Download every table as one JSON file — keep it somewhere safe (e.g. a shared drive) in case anything ever needs to be restored.

Full Backup

Includes items, opening balances, in/out history, orders, duplicated-item mappings, sachets, recipes, sachet in/out history, and the CSV import log.

Restoring

This app doesn't restore a backup automatically — that's intentional, to avoid accidentally overwriting live data. If you ever need to restore, open the JSON file and re-insert the rows you need through Supabase's Table Editor, or ask for help turning it back into SQL.
